# egovSimple
전자정부 프레임워크 egovFrameWork Template Project(Simple Homepage) 초기 프로젝트 세팅 - 단순홈페이지 기능 구현시 필수적인 부분만 사용 가능하도록 경량화 된 실행환경 제공
* 제공기능 : 메인페이지, 사용자관리, 공지사항 관리, 게시판 관리, 안내 관리
